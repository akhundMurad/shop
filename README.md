<h1>Online Shop on DRF</h1>

<h4>Commands:</h4>
Build the project (docker-compose build):<br>

`make build`

Up the project (docker-compose up -d):<br>
`make up`

Run tests:<br>
`make test`

<h4>Envfile structure:</h4>

```
DJANGO_SETTINGS_MODULE=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_HOST=
POSTGRES_PORT=
SECRET_KEY=
```
